1chan-Chat-Controller
=====================

Скрипт управления чатами Колчана.

Умеет работать с вордфильтром, сохраняет пароли разных чатов, имеет небольшой хелп.

Использует jQuery.

========

* Блокировка обхода вордфильтра
* Счетчик постов
